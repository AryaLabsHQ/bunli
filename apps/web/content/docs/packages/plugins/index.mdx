## Available Plugins

### [@bunli/plugin-config](/docs/packages/plugins/config)

Loads and merges configuration from multiple sources including JSON files, RC files, and user config directories.

```bash
bun add @bunli/plugin-config
```

### [@bunli/plugin-ai-detect](/docs/packages/plugins/ai-detect)

Detects AI coding assistants and agents from environment variables, providing context about the execution environment.

```bash
bun add @bunli/plugin-ai-detect
```

### [@bunli/plugin-completions](/docs/packages/plugins/completions)

Generates shell completions (bash, zsh, fish, PowerShell) for your CLI commands.

```bash
bun add @bunli/plugin-completions
```

### [@bunli/plugin-mcp](/docs/packages/plugins/mcp)

Exposes your CLI commands as MCP (Model Context Protocol) tools for AI integration.

```bash
bun add @bunli/plugin-mcp
```

### [@bunli/plugin-skills](/docs/packages/plugins/skills)

Syncs your CLI commands as skill files for AI coding agents (Claude Code, Cursor, Cline, and 30+ others).

```bash
bun add @bunli/plugin-skills
```

## Quick Start

```typescript
import { createCLI } from '@bunli/core'
import { configMergerPlugin } from '@bunli/plugin-config'
import { aiAgentPlugin } from '@bunli/plugin-ai-detect'

const cli = await createCLI({
  name: 'my-cli',
  plugins: [
    configMergerPlugin(),
    aiAgentPlugin({ verbose: true })
  ]
})
```
