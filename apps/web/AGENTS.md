# apps/web

**Next.js documentation site for bunli.**

## OVERVIEW

Next.js app with Fumadocs for MDX-based docs.

## WHERE TO LOOK

| Task | Location |
|------|----------|
| Pages | `app/` |
| Components | `components/` |
| Content | `content/` |
| Config | `next.config.mjs` |

## COMMANDS

```bash
cd apps/web
bun run dev    # Development server
bun run build  # Production build
```
