# @bunli/utils

**Utility functions for Bunli CLIs.**

## OVERVIEW

Provides prompts, spinners, colors, and validation utilities.

## WHERE TO LOOK

| Task | Location |
|------|----------|
| Main exports | `src/index.ts` |
| Prompt utilities | `src/prompt.ts` |
| Spinner | `src/spinner.ts` |
| Colors | `src/colors.ts` |
| Validation | `src/validation.ts` |

## EXPORTS

- `prompt`, `confirm`, `select`, `multiselect`, `password` - Interactive prompts
- `spinner` - Loading spinner
- `colors` - Terminal colors
- `validate*` - Validation utilities
